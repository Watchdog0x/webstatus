# ADD URLS to test 

you can add urls to test in her 
```go
urls := []string{}
```
## example 

```go
urls := []string{"https://google.com", "https://facebook.com", "https://go.dev"}
```

## Output

```
URL                                      Status code
───────────────────────────────────────────────────────
https://go.dev                           200       
https://google.com                       200       
https://facebook.com                     200   
```
